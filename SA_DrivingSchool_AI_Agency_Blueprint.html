The Opportunity

Driving schools in South Africa are high-friction, cash-heavy businesses run almost entirely by one person who is physically unavailable for most of the working day — because they're in a car. Every hour they're teaching is an hour they're not answering calls, replying to WhatsApp, or confirming bookings. That gap is the business.

Install Docker Compose sudo apt install docker-compose-plugin -y # 3. Create n8n directory mkdir ~/n8n-stack && cd ~/n8n-stack # 4. Pull docker-compose.yml (from your setup files) # 5. Start everything docker compose up -d # n8n now runs at http://YOUR_EC2_IP:5678 # Add SSL via Caddy or Certbot for HTTPS (required for PayFast webhooks)

⚠️ AWS Security Group — Open These Ports

In your EC2 Security Group, ensure port 5678 (n8n) and port 443 (HTTPS) are open to 0.0.0.0/0. Without HTTPS, PayFast and Meta webhooks will refuse to connect.

Section 05

Voice Agent — Deep Dive

The voice agent is the premium tier of the offer. Most "AI chatbot" agencies don't include phone handling. This is your differentiator. A driving school owner who gets told "your AI will answer every single call, even at midnight" — that's a story they'll tell other school owners.

Why Vapi.ai (not Bland/Retell)

  • Bland AI: $0.14/min, $299/month minimum — too expensive for SA
  • Retell AI: $0.07/min base but loads to $0.13–$0.31 when you add LLM + TTS
  • Vapi.ai: $0.05/min platform + $0.01/min Twilio = ~$0.06/min total
  • Vapi has the best native n8n integration (webhook tool calling)
  • No monthly platform fee — pure pay-as-you-go

SA Phone Setup

  • Get a Twilio SA local number: $1.50/month
  • Import that number into Vapi dashboard
  • Client does a call forward on their existing number → Twilio number
  • No SIM swap needed. Client keeps their number visibly
  • Inbound call cost via Twilio SA: ~$0.01/min (~18c/min)
  • At 50 calls × 4 min avg = ~R36/month in call costs

Voice Agent System Prompt (Template)

Vapi Assistant — System Prompt
You are the automated booking assistant for [School Name], a driving school based in [City], South Africa. Your job is to help students book driving lessons, get pricing information, and schedule their lessons with an instructor.

You speak in a calm, friendly, South African English tone. You do not use American expressions. Be warm but efficient — students are usually young and slightly nervous.

When a student calls: (1) Greet them and ask what they need. (2) If booking — ask Code 8 or Code 10, automatic or manual, and their preferred days. (3) Use the check_availability tool to look up free slots. (4) Confirm the booking and tell them you're sending a WhatsApp with the deposit link. (5) If they have a complaint, test failure, or refund request — tell them the instructor will call them back and use the escalate_to_owner tool.

Never make up availability. Only quote slots the check_availability tool returns. Never discuss prices you're unsure of — say "I'll have the instructor confirm that with you."
